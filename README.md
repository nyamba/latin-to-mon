Латин үсгээр галиглаж бичсэнийг крилл үсэг рүү хөрвүүлэгч.

## Хэрэглээ

пакежаа npm сангаас татаж суулгана.

```bash
yarn add latin-to-mon
```

жишээ код:

```js
const latin2mon = require("latin-to-mon");
console.log(latin2mon("namyyn")); // "намуун" гэж хэвлэнэ
```
